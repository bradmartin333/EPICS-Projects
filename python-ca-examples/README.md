# Python Channel Access Examples

This directory contains Python scripts demonstrating Channel Access (CA) client capabilities with the MyProject IOC

## Setup

1. Install uv with PowerShell: `powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"`
1. `uv sync`
1. Start MyProject IOC

## Usage

Run scripts with `uv run script.py`
